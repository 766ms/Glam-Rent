🧪 Test del Panel de Administración

📋 Checklist de Verificación

1. ¿El servidor está corriendo?

⏳ Verificando...

2. ¿Existe el usuario admin?

⏳ Verificando...

3. ¿Puedes iniciar sesión?

Credenciales de Admin:
📧 Email: admin@glamrent.com
🔑 Contraseña: admin123

4. ¿Ves el botón "Admin"?

Después de iniciar sesión como admin, deberías ver un botón "🔧 Admin" en la barra de navegación superior.

Si no lo ves, verifica que:

  • Hayas iniciado sesión
  • Usaste las credenciales de admin correctas
  • El usuario tenga es_admin = True

5. ¿Funciona el panel?

Al hacer clic en "Admin" deberías ver:

  • ✅ Modal del panel de administración
  • ✅ Dos pestañas: "Pedidos" y "Productos"
  • ✅ Estadísticas en cards con fondo rosa
  • ✅ Lista de pedidos o productos según la pestaña

🔧 Solución de Problemas

❌ No veo el botón "Admin"

Solución:

  1. Cierra sesión
  2. Vuelve a iniciar sesión con: admin@glamrent.com / admin123
  3. El botón debería aparecer automáticamente

❌ El botón aparece pero no pasa nada al hacer clic

Solución:

  1. Abre la consola del navegador (F12)
  2. Ve a la pestaña "Console"
  3. Busca errores en rojo
  4. Recarga la página con Ctrl + Shift + R

❌ Se abre el modal pero está vacío

Solución:

  1. Verifica que el servidor esté corriendo
  2. Abre la consola (F12) y busca errores de API
  3. Verifica que haya productos y pedidos en la base de datos

❌ No puedo cambiar el stock

Solución:

  1. Cambia el número en el input
  2. Haz clic en el botón "Actualizar"
  3. Deberías ver una notificación de éxito
  4. La página se recargará automáticamente
